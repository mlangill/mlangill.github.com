Introduction

In this lab, you will practice viewing protein structures and structural alignments.

Learning Objectives

  1. Get structures from the PDB.

  2. Use VMD to produce informative figures.

  3. Search for structural analogs in the PDB database.

  4. Identify argument for and against the assignment of homology between a viral protein and other structural analogs.

Step 1: Install VMD

VMD is freely available software for vizualizing protein structures. You need to download (Windows, MacOSX, or Linux) and then install the software.

Step 2: Fetching the HA structure

Go to the NCBI website.

Search for a structure using the PDB key 1HTM as search argument.  Follow the link to the NCBI entry, then onto the PDB database (PDB ID hyperlink at the top of the page).

You should be directed to the PDB itself. Near the top of the "structure summary" page, there are three items. Download the structure into a local folder as a PDB (text). 

By opening the 1HTM.pdb file in a text editor, answer the following questions:

Question 1: What laboratory techinque was used to solve the structure?

Question 2: What is the source taxon id and name for this structure?

Question 3: What is type of element is the first atom in the structure and what is its X, Y, Z coordinates?


Launch VMD from the START menu on the desktop. Click on the menu File\New molecule...  Click on Browse, select the PDB file then click on Load.

Click on Graphics\Representations... Select the draw method as "NewCartoon" and the color scheme to "chain".

Click on "Create Rep".  Set the Select atom field to "chain B".

Experiment with orientation, rendering and color scheme as to clearly represent the overall architecture of this protein. The chain B should be proeminent. Rotating is done with the left or middle mouse button. Rotate mode is triggered by the key 'r', translate by 't', guess which key trigger the scale mode. The wheel mouse works as well for scaling.

When you are happy with what you see on  screen. Click on  VMD main\File\Render... Change the filename and take a snapshot of the screen. The image should open in MS Paint, save the bmp into your working directory. The axis widget can be turned off with Display\Axes...
